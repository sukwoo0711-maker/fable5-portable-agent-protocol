# Pre-Edit Checklist

- [ ] Current request and done condition are clear.
- [ ] Relevant project instructions were checked.
- [ ] Target code was read, not inferred from filename alone.
- [ ] Caller, dependency, or test context was inspected when relevant.
- [ ] Existing local pattern was identified.
- [ ] Current status/diff was inspected when VCS exists.
- [ ] Generated, vendor, lockfile, and artifact boundaries were checked.
- [ ] Approval is not required, or approval was obtained.
- [ ] Planned change is the smallest responsible change.
